# Customer Behavior Analysis

## Overview

This project demonstrates an end-to-end data analytics workflow using Python, MySQL, and Power BI. The objective is to clean and analyze customer shopping behavior data, extract meaningful business insights using SQL, and present the findings through an interactive Power BI dashboard.

---

## Dataset

The dataset contains customer shopping behavior information, including:

* Customer demographics
* Purchase amount
* Product category
* Review ratings
* Subscription status
* Shipping type
* Discount usage
* Previous purchases

The raw dataset is cleaned using Python before being analyzed in MySQL and visualized in Power BI.

---

## Tools & Technologies

* **Python** – Data loading, exploratory data analysis (EDA), and data cleaning
* **Pandas** – Data manipulation and preprocessing
* **MySQL** – Business insights using SQL queries
* **Power BI** – Interactive dashboard creation

---

## Project Workflow

### 1. Data Loading

* Imported the raw CSV dataset using Python.
* Examined the dataset structure and data types.

### 2. Exploratory Data Analysis (EDA)

* Explored missing values.
* Checked duplicate records.
* Analyzed data distributions.
* Identified inconsistencies and data quality issues.

### 3. Data Cleaning

* Removed duplicates.
* Handled missing values.
* Standardized column names and data formats.
* Created a cleaned dataset for further analysis.

### 4. SQL Analysis (MySQL)

Business questions answered include:

* Revenue generated by different customer segments.
* Impact of discounts on purchasing behavior.
* Top-rated and most purchased products.
* Subscription-based customer spending.
* Revenue contribution by different age groups.
* Customer segmentation based on previous purchases.
* Product performance across categories.

### 5. Power BI Dashboard

Built an interactive dashboard to visualize key business metrics, including:

* Total Revenue
* Average Purchase Amount
* Revenue by Gender
* Revenue by Age Group
* Product Category Performance
* Subscription Analysis
* Shipping Preferences
* Interactive filters and slicers for deeper exploration

---

## Dashboard

The Power BI dashboard provides an interactive overview of customer purchasing behavior, allowing users to filter data and explore business performance across different customer segments, product categories, and purchasing patterns.

---

## Results

The analysis provides valuable business insights, including:

* Customer segments contributing the highest revenue.
* Spending patterns of subscribed vs. non-subsscribed customers.
* Most popular and highest-rated products.
* Effect of discounts on customer purchases.
* Revenue trends across age groups and shipping methods.

## How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/your-repository.git
   ```

2. Navigate to the project folder:

   ```bash
   cd your-repository
   ```

3. Install the required Python packages:

   ```bash
   pip install -r requirements.txt
   ```

4. Run the Python script:

   ```bash
   python main.py
   ```

5. Import the cleaned dataset into MySQL and execute the SQL queries.

6. Open the Power BI (`.pbix`) file to explore the interactive dashboard.
